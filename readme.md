# SumoBalls



SumoBalls es un juego competitivo local para dos jugadores donde cada jugador controla una bola. El objetivo es empujar al oponente fuera del área usando la fuerza y el movimiento.



## Cómo jugar



- **Jugador 1**

&nbsp; - Moverse: `W A S D`

&nbsp; - Saltar: `W`



- **Jugador 2**

&nbsp; - Moverse: Flechas del teclado

&nbsp; - Saltar: `↑`



Ambos jugadores pueden moverse de izquierda a derecha y saltar para ganar ventaja en los choques.



## Objetivo



Empuja al oponente fuera del escenario para ganar la partida.
